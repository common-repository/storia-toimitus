WooCommerce integration plugin for WordPress, designed for the logistic customers of Storia Oy.

== Description ==
This plugin integrates WooCommerce with Storia logistics.

Storia is located in Finland. We offer logistics services to our customers. For more information, please visit our website.

== Installation ==
1. Upload the `storia-toimitus` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'API Key Generation' submenu, found under Storia tab.
4. Create a new API key and deliver it to Storia. You only need to do this step once, or if your API key has been compromised.
